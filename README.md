# Google Translate IP

```Golang``` 版查找，测速谷歌翻译可用  ```IP``` 地址

## 使用方法

1. 下载 ```exe``` 程序和 ```ip.txt``` 文件，放置于同一目录中。
2. 运行 ```windows-main.exe``` 。

## 获取可用IP地址列表
